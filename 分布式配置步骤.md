# 配置分布式有哪些要点
1. 找一台Windows Server版本的机器，最低2016.作为Domain Server(域服务器);
2. 将用到的结点机器的账户，设置为相同的密码，然后都加入域；
3. 将MPITest.exe拷贝到每一个节点计算机，而且其路径相同。 或者拷贝到同一网络共享路径，然后确保每个节点都对该路径有访问权限。//PS: “映射网络驱动器”工具，也可以将本地路径映射为一个虚拟盘;
4. 设置启动参数-hosts;
5. 在各个节点计算机上，设置允许8677、8676端口通过防火墙； 将MPITest.exe添加到防火墙白名单。或者直接关闭防火墙.
6. MsMpiLaunchSvc显示不了界面。可以关闭该服务，以启动smpd.exe程序来代替服务。
